# Manga Reader

A minimalist web app for searching and reading manga using the [MangaDex API](https://api.mangadex.org).  
Built to provide a distraction-free manga reading experience with fast search and clean chapter viewing.

## Tech Stack

- **Backend:** Express.js
- **Templating Engine:** EJS
- **Language:** TypeScript
- **Data Source:** MangaDex API
